So this is a small project I created to review some aspects of OOP.

I reviewed the following topics:


- Class
- Object
- Constructor
- Encapsulation
- Inheritance
- Polymorphism (Overriding and Overloading)
-  Abstraction
- Interfaces


I followed a YouTube tutorial created by ForrestKnight:  
https://www.youtube.com/watch?v=TiccevwEVe8

The idea was to create a simple system similar to an inventory in video games.

The Inventory class manages items. Item is a more general superclass, which is extended by Weapon and Fruit.

I used two different design approaches, following the tutorial mentioned above: abstract classes (not being used at the moment) and interfaces.
